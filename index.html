CSharpGuidelinesAnalyzer

Reports diagnostics for C# coding guidelines that are not already covered by Resharper.

Download this project as a .zip file Download this project as a tar.gz file

This Visual Studio analyzer supports you in making your code comply with the C# coding guidelines at CSharpGuidelines. Note that many guidelines are already covered by Resharper, which are not implemented here.
